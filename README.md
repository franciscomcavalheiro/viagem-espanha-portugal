# Roteiro Espanha e Portugal 2026

Site interativo apresentando um roteiro de viagem premium pela Espanha e Portugal.

## 📋 Conteúdo

- **Cidades Principais**: Lisboa, Porto, Madrid, Barcelona, Sevilha e Granada
- **Experiências em Vinícolas**: Vale do Douro e Rioja
- **Hospedagem Premium**: Hotéis de luxo selecionados
- **Experiências Exclusivas**: Tours guiados, gastronomia e atividades culturais

## 🚀 Como Usar

1. Abra o arquivo `index.html` em um navegador web
2. Navegue pelas diferentes seções do roteiro
3. Explore as imagens e informações sobre cada destino

## 📁 Estrutura

```
├── index.html          # Página principal
├── assets/            # JavaScript e CSS compilados
├── images/            # Imagens dos destinos
└── server.py          # Servidor Python para desenvolvimento
```

## 💻 Desenvolvimento Local

Para servir o site localmente com Python:

```bash
python3 server.py
```

Acesse `http://localhost:8080` no seu navegador.
